# Faxina Confiável 🧹

Site criado para ajudar donas de casa e famílias a encontrarem diaristas de confiança para serviços domésticos, oferecendo praticidade e segurança no agendamento.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- Bootstrap


## 📑 Funcionalidades
- Página inicial com chamada principal e botão de agendamento
- Página "Sobre" com apresentação da profissional
- Página "Serviços" com cards de serviços oferecidos
- Página "Contato" com formulário e links para WhatsApp e redes sociais
